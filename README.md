# @ianwalter/faygit
> Generate a mock git repository: faygit til you make it!

[![npm page][npmImage]][npmUrl]
[![CI][ciImage]][ciUrl]

# Installation

Globally:

```console
yarn global add @ianwalter/faygit
```

Or as a dev dependency:

```console
yarn add @ianwalter/faygit --dev
```

## Usage

```console
faygit [directory]
```
